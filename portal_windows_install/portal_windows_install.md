# Windows 任务计划管理安装

## 概述
用于自动化管理Windows任务计划，实现`portal.exe`程序的开机后台执行。

###  权限管理
自动检测管理员权限，非管理员时自动请求提权

### 文件部署
目标路径	说明
 - C:\Program Files\portal\	主安装目录
 - portal.exe	主程序副本
 - portal.conf	配置文件模板

### 主菜单功能
 - 添加任务计划名称为auto_portal（添加后立即执行任务） → 创建SYSTEM权限任务(系统开机自启动) → 自动复制文件到系统目录
 - 删除现有任务计划
 - 显示任务计划状态
 - 退出程序
